# Storage su GCP

GCP offre soluzioni di storage per ogni tipo di workload. Cloud Storage (GCS) è il servizio di object storage fondamentale, usato come backend per quasi ogni architettura GCP: Terraform state, data lake, log export, artefatti CI/CD, dataset ML.
