# Fondamentali di Networking

Comprendere i fondamentali del networking è prerequisito essenziale per qualsiasi ingegnere DevOps: ogni sistema distribuito, ogni pipeline CI/CD, ogni deployment cloud si appoggia su questi protocolli e meccanismi. Senza una solida base teorica e pratica, il troubleshooting di problemi di connettività, latenza o sicurezza diventa un processo alla cieca. Questa sezione copre i protocolli e i concetti su cui è costruita l'intera infrastruttura moderna.
